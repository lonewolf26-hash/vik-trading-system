Premarket Scanners — Run 8:00–9:25 AM ET
01
Premarket Parabolic Gapper
Finds stocks gapping up 50%+ premarket on 5x+ relative volume under $5. Builds your short watchlist before the open. This is how you find RMSG, SNAL, SNYR type setups before 9:30.
Aggregation: Day Run: 8:00–9:25 AM ET Price: $0.50–$5.00 Gap: 50%+ Rel Vol: 5x+
ThinkScript — paste in Stock Hacker
# 01 — PREMARKET PARABOLIC GAPPER # Finds stocks gapping 50%+ premarket on 5x+ relative volume # Run: 8:00 AM – 9:25 AM ET | Aggregation: Day def priorClose = close(period = AggregationPeriod.DAY)[1]; def price = close; def gapPct = (price - priorClose) / priorClose * 100; def avgVol = Average(volume, 20); def relVol = volume / avgVol; plot scan = price > 0.50 and price < 5.00 and gapPct >= 50 and relVol >= 5;
Setup in TOS
  1. Scan tab → Stock Hacker
  2. Add Study Filter → Study → Edit → paste code
  3. Set aggregation to Day
  4. Add UI filter: Last between $0.50–$5.00, Exchange: NASDAQ/NYSE
  5. Sort results by % Change descending
  6. Save tab as "PM Gapper"
Intraday Scanners — Run After 9:30 AM ET
02
Intraday VWAP Extension Fade
Flags stocks trading 30%+ above VWAP with slowing momentum candles. Catches the exhaustion window when a parabolic move is fading and the short entry is setting up.
Aggregation: 1 min Run: 9:30–11:30 AM ET VWAP Extension: 30%+ Rel Vol: 10x+ Momentum Slowing
ThinkScript — paste in Stock Hacker
# 02 — INTRADAY VWAP EXTENSION FADE # Flags stocks 30%+ above VWAP with slowing momentum # Run: 9:30 AM – 11:30 AM ET | Aggregation: 1 min def price = close; def vwapVal = VWAP(); def vwapExt = (price - vwapVal) / vwapVal * 100; def avgVol = Average(volume, 20); def relVol = volume / avgVol; def range = high - low; def avgRange = Average(range, 5); def momentumSlowing = range < avgRange * 0.70; plot scan = price > 0.50 and price < 5.00 and vwapExt >= 30 and relVol >= 10 and momentumSlowing;
What to look for when triggered
  1. Red candle rejecting HOD on the 1-min chart
  2. Volume drying up on each upside push
  3. Lower highs forming — each push weaker than last
  4. Price losing VWAP support on a pullback
03
Extreme Volume Spike / Halt Detector
Flags stocks with 50x+ average volume — likely halted or just resumed. Gives early warning before RMSG-type situations so you can plan your entry and stop before the chaos hits.
Aggregation: 1 min Run: All day Rel Vol: 50x+ Price: $0.50–$5.00
ThinkScript — paste in Stock Hacker
# 03 — EXTREME VOLUME SPIKE / HALT DETECTOR # Flags stocks 50x+ average volume — halt candidate or resume # Run: All day | Aggregation: 1 min def price = close; def avgVol = Average(volume, 20); def relVol = volume / avgVol; plot scan = price > 0.50 and price < 5.00 and relVol >= 50;
Pre-trade checklist when triggered
  1. Check borrow availability on IBKR before entering
  2. Confirm catalyst is weak — no real news, vague PR
  3. Set hard stop immediately at entry — above HOD
  4. Plan halt resume exit in advance — up or down

04
Low Float 3-Day Momentum Runner
Finds low float stocks under $10 that have risen 50%+ over the last 3 trading days. Surfaces multi-day runners for short fade or continuation plays.
Aggregation: Day Float: under 20M Price: under $10 3-Day Gain: 50%+
ThinkScript — paste in Stock Hacker
# 04 — LOW FLOAT 3-DAY MOMENTUM RUNNER # Float < 20M, Price < $10, risen 50%+ over last 3 trading days # Aggregation: Day # Add manually: Fundamental filter → Shares Float < 20,000,000 input max_price = 10.00; input min_gain_3d = 50.0; input rel_vol_min = 1.5; input avg_vol_length = 20; def price_ok = close < max_price and close > 0.20; def avg_vol = Average(volume, avg_vol_length); def rel_vol = volume / avg_vol; def vol_ok = rel_vol >= rel_vol_min; def gain_3d = ((close - close[3]) / close[3]) * 100; def is_runner = gain_3d >= min_gain_3d; plot scan = price_ok and vol_ok and is_runner;
Setup in TOS

  2. Add Fundamental filter: Shares Float → Less Than → 20,000,000
  3. Set aggregation to Day
  4. Save tab as "3-Day Runner"

Aggregation: Day Float: under 30M Price: $0.20–$15 Single-Day Gain: 20%+ Lookback: 5 days

06
Failed Breakout / HOD Rejection
Finds stocks that tagged a new 10-day high intraday but closed below it — a classic rejection candle. Best short entry signal on parabolic stocks losing momentum at the highs.
Aggregation: Day Price: under $15 Rel Vol: 1.5x+ HOD Rejection Bearish Close
ThinkScript — paste in Stock Hacker
# 06 — FAILED BREAKOUT / HOD REJECTION # Stock tagged 10-day high but closed back below it # Bearish close in lower 40% of day's range + high rel vol # Aggregation: Day input max_price = 15.00; input lookback_high = 10; input rel_vol_min = 1.5; input avg_vol_length = 20; def price_ok = close < max_price; def avg_vol = Average(volume, avg_vol_length); def rel_vol = volume / avg_vol; def vol_ok = rel_vol >= rel_vol_min; def resistance = Highest(high[1], lookback_high); def tagged_resistance = high >= resistance; def closed_below = close < resistance; def candle_range = high - low; def close_pos = (close - low) / candle_range; def bearish_close = candle_range > 0 and close_pos <= 0.40; plot scan = price_ok and vol_ok and tagged_resistance and closed_below and bearish_close;
07
Parabolic Fade / Bearish Reversal Candle
Finds stocks up 50%+ in 3 days that are showing the first bearish reversal candle — red candle with upper wick, extended above the 10-day MA. Best for timing short entries on exhausted runners.
Aggregation: Day Price: under $15 3-Day Gain: 50%+ Extended above MA Bearish Reversal Candle
ThinkScript — paste in Stock Hacker
# 07 — PARABOLIC FADE / BEARISH REVERSAL CANDLE # Up 50%+ in 3 days, extended above MA, red candle with upper wick # Best for timing short entries on exhausted runners # Aggregation: Day input max_price = 15.00; input parabolic_pct = 50.0; input ma_length = 10; input ma_ext_pct = 30.0; input rel_vol_min = 1.5; input avg_vol_length = 20; def price_ok = close < max_price; def avg_vol = Average(volume, avg_vol_length); def rel_vol = volume / avg_vol; def vol_ok = rel_vol >= rel_vol_min; def gain_3d = ((close - close[3]) / close[3]) * 100; def is_parabolic = gain_3d >= parabolic_pct; def ma = Average(close, ma_length); def pct_above_ma = ((close - ma) / ma) * 100; def is_extended = pct_above_ma >= ma_ext_pct; def is_red = close < open; def candle_range = high - low; def upper_wick = high - Max(close, open); def wick_ratio = if candle_range > 0 then upper_wick / candle_range else 0; def bearish_candle = is_red and wick_ratio >= 0.25; plot scan = price_ok and vol_ok and is_parabolic and is_extended and bearish_candle;
08
Breakdown Below Support
Finds stocks breaking below their 10-day support low while below both the 9 and 20 EMA, with a red candle and elevated volume. Confirms a structural breakdown — not just a pullback.
Aggregation: Day Price: under $15 Below 9 + 20 EMA 10-Day Support Break Red Candle
ThinkScript — paste in Stock Hacker
# 08 — BREAKDOWN BELOW SUPPORT # Close below 9 EMA + 20 EMA + 10-day support low # Red candle + elevated volume = structural breakdown # Aggregation: Day input max_price = 15.00; input fast_ema = 9; input slow_ema = 20; input support_days = 10; input rel_vol_min = 1.5; input avg_vol_length = 20; def price_ok = close < max_price; def avg_vol = Average(volume, avg_vol_length); def rel_vol = volume / avg_vol; def vol_ok = rel_vol >= rel_vol_min; def ema9 = ExpAverage(close, fast_ema); def ema20 = ExpAverage(close, slow_ema); def below_emas = close < ema9 and close < ema20; def ema9_down = ema9 < ema9[1]; def ema20_down = ema20 < ema20[1]; def emas_falling = ema9_down and ema20_down; def support = Lowest(low[1], support_days); def breakdown = close < support; def red_candle = close < open; plot scan = price_ok and vol_ok and below_emas and emas_falling and breakdown and red_candle;
09
High Short Interest + Price Weakness
Finds stocks in a downtrend (below 20 EMA for 3+ days) making lower highs and lower lows with a weak close. Add a manual short interest filter for maximum effectiveness.
Aggregation: Day Price: under $15 Below 20 EMA: 3+ days Lower High + Lower Low Add: SI% > 20% manually
ThinkScript — paste in Stock Hacker
# 09 — HIGH SHORT INTEREST + PRICE WEAKNESS # Downtrend: below 20 EMA for 3+ days # Lower high + lower low today vs yesterday # Weak close in bottom 30% of range + elevated volume # Add manually: Fundamental → Short Interest % of Float > 20 # Aggregation: Day input max_price = 15.00; input ema_length = 20; input downtrend_days = 3; input rel_vol_min = 1.5; input avg_vol_length = 20; def price_ok = close < max_price; def avg_vol = Average(volume, avg_vol_length); def rel_vol = volume / avg_vol; def vol_ok = rel_vol >= rel_vol_min; def ema20 = ExpAverage(close, ema_length); def below_ema_d0 = close[0] < ema20[0]; def below_ema_d1 = close[1] < ema20[1]; def below_ema_d2 = close[2] < ema20[2]; def in_downtrend = below_ema_d0 and below_ema_d1 and below_ema_d2; def lower_high = high < high[1]; def lower_low = low < low[1]; def lower_hl = lower_high and lower_low; def candle_range = high - low; def close_pos = if candle_range > 0 then (close - low) / candle_range else 0; def weak_close = close_pos <= 0.30; plot scan = price_ok and vol_ok and in_downtrend and lower_hl and weak_close;
Add Short Interest Filter
  1. After pasting script, click Add Study Filter again
  2. Choose Fundamental → Short Interest % of Float
  3. Set to Greater Than → 20
  4. This confirms heavy short interest is already pressing the stock
Chart Studies — Paste on Chart, Not in Scanner
10
Momentum Breakout + VWAP Alert
Chart study that fires an alert when a stock crosses above the prior bar high while above VWAP with 2.5x+ relative volume. Paints an arrow on the chart and sounds an alert tone.
Paste on Chart Charts → Edit Studies Rel Vol: 2.5x+ Above VWAP Breakout Candle
ThinkScript — paste as Chart Study
# 10 — MOMENTUM BREAKOUT + VWAP ALERT # Fires when: price crosses above prior bar high + above VWAP + 2.5x rel vol # Paste on chart: Charts → Edit Studies → Create New Study # Then: right-click chart → Create Alert → Study Alert input rvol_threshold = 2.5; input lookback = 20; def avgVol = Average(volume, lookback); def relVol = volume / avgVol; def vwap = VWAP(); def priorHigh = High[1]; def breakout = Close crosses above priorHigh; def aboveVWAP = Close > vwap; def highRelVol = relVol >= rvol_threshold; def signal = breakout and aboveVWAP and highRelVol; plot Alert = signal; Alert.SetPaintingStrategy(PaintingStrategy.BOOLEAN_ARROW_UP); Alert.SetDefaultColor(Color.CYAN); Alert.SetLineWeight(3); Alert(signal, "Momentum Breakout", Alert.BAR, Sound.Ding);
Wiring the Alert
  1. Paste as a chart study: Charts → Edit Studies → Create New Study
  2. Right-click chart → Create Alert
  3. Alert Type → Study Alert → select this study
  4. Condition: Alert is true
  5. Check Sound, Pop-up, optionally Email/SMS
11
ATR-Based R/R Painter
Paints the stop level (1x ATR below entry) and 2:1 target level directly on the chart as dashed lines. Visual reference for risk/reward without manual calculation.
Paste on Chart ATR Period: 14 Stop: 1x ATR below Target: 2x stop distance
ThinkScript — paste as Chart Study
# 11 — ATR-BASED R/R PAINTER # Paints stop level and 2:1 target on chart as dashed lines # Paste on chart: Charts → Edit Studies → Create New Study input atr_period = 14; input atr_stop_mult = 1.0; # Stop = 1x ATR below entry input rr_ratio = 2.0; # Target = 2x the stop distance def atr = ATR(atr_period); def stopLevel = Close - (atr * atr_stop_mult); def targetLevel = Close + (atr * atr_stop_mult * rr_ratio); plot StopRef = stopLevel; StopRef.SetDefaultColor(Color.RED); StopRef.SetStyle(Curve.SHORT_DASH); StopRef.SetLineWeight(1); plot TargetRef = targetLevel; TargetRef.SetDefaultColor(Color.GREEN); TargetRef.SetStyle(Curve.SHORT_DASH); TargetRef.SetLineWeight(1);

  2. 9:30 AM — Switch to Scanner 02 (VWAP Extension Fade) to time short entries on gap-ups from watchlist.
